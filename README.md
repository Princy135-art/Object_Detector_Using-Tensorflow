# Object Detection using TensorFlow
This project performs object detection on images using a pre-trained TensorFlow model.  
It identifies objects like animals, people, and everyday items and displays detection results.
## Features
- Detect objects in images
- Uses TensorFlow Hub pre-trained model
- Supports custom images
- Displays detection results
## Requirements
- Python
- TensorFlow
- TensorFlow Hub
- OpenCV
- Matplotlib
